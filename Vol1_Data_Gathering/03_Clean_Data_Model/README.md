# Standardized Unified Data Model

This folder hosts the final consolidated and programmatically cleaned dataset outputted directly from Power Query ETL pipeline.

*   **Single Source of Truth:** Consolidated Marjane, Carrefour, and LabelVie files into one master sheet with full branch traceability.
*   **Audit-Ready:** Zero formatting errors, sanitized product text naming, and unified chronological tracking.
*   **Ready for BI:** This clean database acts as our core analytical backend, supporting all downstream operational KPIs, Pivot Tables, and dynamic executive dashboards.
