## 🎯 Executive Summary
This end-to-end data analytics project bridges the gap between raw, unstructured data and actionable strategic marketing decisions. It explores the monthly sales performance, market penetration, and supply chain dynamics of two major FMCG products: **Dove Nutritive** and **Sunsilk Expert**. 

The primary objective is to build a robust ETL pipeline, model the data for interactive dashboards, and conduct deep-dive exploratory data analysis (EDA) to uncover variance, volume vs. value relationships, and hidden logistical bottlenecks.

---

## 🏗️ Project Architecture & Methodology

This repository is structured into sequential volumes, simulating a real-world data engineering and analytics workflow:

### 📁 [Vol. 1: Data Gathering & ETL](Vol1_Data_Gathering/)
* **Objective:** Extract, Transform, and Load (ETL) raw sales data.
* **Key Tools:** Power Query, M Code.
* **Highlights:** Developed custom M scripts for hybrid date formatting and revenue correction to ensure a pristine data model.

### 📁 [Vol. 1 - Extension: Dashboard Architecture](Vol1_Extension_Dashboard_Architecture/)
* **Objective:** Design the front-end interactive interface.
* **Key Tools:** Excel Advanced Data Modeling, Pivot Tables.
* **Highlights:** Constructed a dynamic, user-friendly dashboard integrating the cleaned data model for high-level management overview.

### 📁 [Vol. 2: Exploratory Data Analysis (EDA)](Vol2_Exploratory_Data_Analysis_Pandas/)
* **Objective:** Deep statistical analysis and pattern recognition.
* **Key Tools:** Python (Pandas).
* **Highlights:** 
  * Analyzed core KPIs and calculated performance variance.
  * Investigated the "Volume vs. Value" paradox.
  * Segmented the month into tertiles for granular trend analysis, successfully identifying a recurring end-of-tertile supply chain anomaly affecting product availability.
